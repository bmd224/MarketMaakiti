# MarketMaakiti
Market Maakiti est une application Android qui permet aux utilisateurs d'acheter et de vendre des articles avec un chat integré pour leur permettre de discuter directement.
## Fonctionnalités
* Authentification des utilisateurs (inscription et connexion)
* Parcourir et rechercher des articles
* Voir les détails des articles
* Ajouter des articles aux favoris
* Publier de nouvelles annonces
* Gérer vos propres annonces
* Messagerie instantanée avec d'autres utilisateurs
* Gestion du profil utilisateur
* Mode sombre
## Technologies
* Kotlin
* Firebase
    * Authentification
    * Firestore
    * Storage
    * Cloud Messaging
* Jetpack Compose
## Démarrage
Pour commencer, vous devez avoir Android Studio installé.
1. Clonez le dépôt
2. Ouvrez le projet dans Android Studio.
3. Créez un fichier `google-services.json` dans le répertoire `app` et ajoutez la configuration de votre projet Firebase.
4. Exécutez l'application
